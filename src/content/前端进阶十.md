## flubber

流畅动画库

npm安装

```shell
npm install flubber
```

html引入

```html
<script src="https://unpkg.com/flubber@0.3.0"></script>
```

使用

```javascript
var flubber = require("flubber"); // Node classic
import { interpolate } from "flubber" // ES6
```
